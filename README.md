# Po-Em-MHLCDS

This repository contains the dataset and code for our paper titled [Help me heal: A reinforced polite and empathetic mental health and legal counseling dialogue system for crime victims](https://ojs.aaai.org/index.php/AAAI/article/view/26685)

## Code
The folder Po-Em-MHLCDS contains the code for the proposed Po-Em-MHLCDS architecture.
